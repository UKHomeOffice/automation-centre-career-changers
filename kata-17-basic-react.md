# Kata 17 - Basic React

## Overview

Building from the previous katas, we're now going to look at React. Although it is more likely that within the Home Office you will be using NextJS, learning the fundamentals of React is essential to understanding how NextJS works under the hood.

## Objectives

Complete the linked sections of the React learning path on Pluralsight to gain a solid understanding of React fundamentals.

You are more than welcome to explore further sections of the learning path if you wish to deepen your knowledge, but the sections listed should give a reasonable overview.
